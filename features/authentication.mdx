## Overview

BioTea implements a secure authentication system using React Native with Expo Router. The authentication flow integrates with a remote API and securely stores user credentials using Expo's SecureStore.

## Authentication Flow

The login process follows these steps:

1. User enters credentials (username and password)
2. Credentials are validated locally for empty fields
3. Request is sent to the authentication API
4. On success, user data is stored in both UserContext and SecureStore
5. User is navigated to the home screen

## Login Implementation

The login screen is implemented in `app/login.tsx:9`. Here's the complete authentication handler:

<CodeGroup>
```typescript Login Handler
const handleLogin = async () => {
    if (username === '' || password === '') {
        Alert.alert('Campos vacíos', 'Por favor, complete todos los campos.');
        return;
    }

    try {
        const requestData = {
            user: username,
            pass: password,
        };

        const response = await axios.post(
            `${apiEndpoints.dev}/login.php`,
            requestData
        );

        if (!response.data.error) {
            // Si el inicio de sesión es exitoso, navega a la pantalla de inicio y pasa los datos del usuario
            setUser(response.data.data);
            await saveSecureData('user', response.data.data);
            router.navigate('(tabs)/(home)/inic');

        } else {
            Alert.alert('Inicio de sesión fallido', response.data.message || "Datos incorrectos, por favor revise su información de inicio de sesión");
        }
    } catch (error) {
        console.log(error);
        Alert.alert('Error', 'Hubo un error al intentar iniciar sesión. Por favor, inténtelo de nuevo más tarde.');
    } finally {
        // Limpiar los campos de entrada después de un intento de inicio de sesión, exitoso o no
        setUsername('');
        setPassword('');
    }
};
```

## API Request Format

The login endpoint expects a POST request with the following structure:

```typescript
const requestData = {
    user: username,
    pass: password,
};

const response = await axios.post(
    `${apiEndpoints.dev}/login.php`,
    requestData
);
```

## Response Handling

The API response is expected to have this structure:

```typescript
{
    error: boolean,
    message?: string,
    data?: {
        nombres: string,
        correo: string,
        // ... other user fields
    }
}
```

<Warning>
The login handler clears the username and password fields in the `finally` block, ensuring credentials are removed from component state after each login attempt, whether successful or not.
</Warning>
